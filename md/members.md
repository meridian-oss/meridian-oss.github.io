# 開発体制

- すべて有志による開発となります.
- コア開発チームのメンバーは毎年更新します.
- 活動や貢献に対する金銭的な報酬はありません.  

## 役割

### コア開発メンバー / Core Maintainer

- コア部分のデザインや実装を行います.  
- コア部分に隣接するモジュールの開発や実装を行います.  
- ターゲットが理解, 再現できる形でプロジェクトを公開します.  

### 応用開発メンバー / Application Maintainer

- Meridianの応用事例を開発・実装し, 公開します.  
- ターゲットが理解, 再現できる形でプロジェクトを公開します.  

### コントリビューター / Contributor

- GitHubのプルリクエストベースで協力いただきます.  

### 開発モニター / Beta Tester

- コア部分を利用し, ユーザーフィードバックを行います.
